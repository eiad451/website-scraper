# Website Scraper Pro 🕵️

أداة متكاملة لسحب المواقع بالكامل، تحليل ملفات APK، سحب قنوات التلغرام، وفحص المسارات المخفية.

## المميزات
- 🌐 سحب مواقع كامل (HTML, CSS, JS, صور)
- 📱 تحليل APK مع فك تجميع وذكاء اصطناعي
- ✈️ سحب قنوات تلغرام (بوت API أو عام)
- 🔍 فحص مسارات مخفية + لوحات إدارة + قواعد بيانات
- 🛡️ تجاوز Cloudflare
- 👥 نظام مستخدمين بعملات واشتراكات
- ⚙️ لوحة إدارة كاملة

## التقنيات
- PHP 8+ (Backend)
- Python 3 (Cloudscraper, Bot)
- JavaScript + CSS (Frontend)
- SQLite (Database)

## التشغيل
```bash
chmod +x server.sh
./server.sh start
```
أو مباشرة:
```bash
php -S 0.0.0.0:8080 -t .
```
